# ghostbsd-gershwin-wrappers
Compiled application wrappers

### Prerequisites

- Gershwin installation

## Build Instructions

### Build
```bash
gmake
```

### Install
```bash
sudo gmake install 
```

### Run
```bash
openapp HelloWorld
```

### Generate new wrappers

```
./generate-wrapper-code.sh Chromium /usr/local/bin/chromium /Users/jmaloney/Downloads/chrome.png
```
